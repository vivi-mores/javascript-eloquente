# JavaScript Eloquente 📚

Repositório dedicado ao estudo e à resolução dos exercícios do livro **Eloquent JavaScript**, de Marijn Haverbeke.

O objetivo é aprofundar os conhecimentos em JavaScript por meio da prática, explorando desde os fundamentos da linguagem até tópicos avançados, como estruturas de dados, funções de alta ordem, orientação a objetos, expressões regulares, módulos, programação assíncrona e Node.js.

## Livro

🌐 Leia gratuitamente online: https://eloquentjavascript.net/

## Estrutura

Cada capítulo possui uma pasta própria contendo as soluções dos exercícios e eventuais anotações.

## Objetivo

- Praticar JavaScript puro (Vanilla JS)
- Desenvolver raciocínio lógico e resolução de problemas
- Consolidar conceitos fundamentais e avançados da linguagem
- Criar um histórico público da evolução nos estudos
